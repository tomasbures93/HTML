Command line interface

What is it? A command-line interface (CLI) is a means of interacting with a computer program by inputting lines of text called command-lines. Knowledge of CLIs is also useful for writing scripts. Programs that have CLIs are generally easy to automate via scripting, since command-lines, being mere lines of text, are easy to specify in code.

HyperText Markup Language

What is it? HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for its appearance.

Basics

SELECT * FROM Production.Product; - Select everything from Table Production.Product
SELECT OrderDate AS [Dates] FROM Production.Product; - Select Column OrderDate from Table Production.Product and give it alias Dates
SELECT ProductID, Name + '(' + ProductNumber + ')', ListPrice - StandardCost FROM Production.Product; - Select ProductID, Name and ProductNumber in one column (CONCATINATE), and substract StandardCost from ListPrice
CAST(ProductID AS varchar(4)) - We are converting/casting one Datatype into another , now we are casting INT into varchar(string) ... But we have to be sure we are able to (varchar into int does not need to work)
TRY_CAST(Size AS integer) - We are trying to cast. If it is possible to cast Size as int, we will get numerical outpit, if it is not able to do it, we will get NULL
CONVERT(varchar(4), ProductID) - Same as CAST but different syntax, mostly used for Date, we can choose different date formats with CONVERT
PARSE('01/01/2021' AS date) - converts the string '01/01/2021' into a date data type
STR(ListPrice) - Converts numerical value into string
ISNULL(MiddleName, 'None') - If our output would be null and we dont want that we can use this function to change all outputs into something we want
SELECT COALESCE ( expression1, expression2, [ ,...n ] ) . evaluates the given expressions in order and returns the first non-NULL value it encounters.


Creating Users

SELECT USER_NAME(); - Who am I ?
CREATE USER < user name > WITHOUT LOGIN; - Creating user without login (only user in database)
CREATE LOGIN < login name > WITH PASSWORD = 'password' - Creating user which will be able to Log in Management studio.
CREATE USER < user name > FOR LOGIN < login name > - Now user can login into Database and fully use it if he has privilegis
EXECUTE AS USER = 'user name' - Switching users in Database
REVERT - Switching to the last user
GRANT < what commands > ON < schema or others > TO user - Granting privilegis to user so they can do stuff
REVOKE - Revoking privilegis


Transactions and Savepoints

BEGIN TRAN - Starting a Transaction
COMMIT - Commiting a Transaction
ROLLBACK - Canceling whole Transaction
SAVE TRAN < name > - SAVEPOINT for ROLLBACK
ROLLBACK TRANSACTION < name > - Canceling to the SAVEPOINT
SELECT @@TRANCOUNT - How much Transaction is open
SELECT XACT_STATE(); - Check if transation is open (doesnt matter how much). State: 1 - Yes , 0 - No, -1 - Transaction is broken
